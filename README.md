# Mobile Application Penetration Testing on DIVA

This project demonstrates practical mobile application penetration testing performed on the DIVA (Damn Insecure Vulnerable App).

## Tools Used
- ADB
- MobSF
- APKTool
- Kali Linux
- Genymotion

## Key Findings
- Insecure data storage vulnerability
- Debug mode enabled
- Exported components
- Dangerous permissions

## Skills Gained
- Android reverse engineering
- Mobile security assessment
- Static analysis
- APK modification and signing
